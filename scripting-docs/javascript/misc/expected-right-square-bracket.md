# <a name="expected-"></a>']'가 필요합니다.
배열 요소에 대 한 참조를 만들었지만 오른쪽 대괄호가 포함 되지 않았습니다. 배열 요소를 참조 하는 모든 식에는 여는 대괄호와 닫는 대괄호를 모두 포함 해야 합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 배열 요소를 참조 하는 식에 오른쪽 괄호를 추가 합니다.
