# <a name="c28137"></a>C28137
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C28137: 변수 인수가 대신 (리터럴) 상수 여야 합니다.  
  
 함수 호출이 필요한 상수(리터럴)가 없는 경우 이 경고가 보고됩니다. 함수에 대한 설명서를 참조하세요.  
  
## <a name="example"></a>예제  
 예를 들어 [ExAcquireResourceExclusiveLite](https://msdn.microsoft.com/library/windows/hardware/ff544351.aspx) 루틴에는 매개 변수에 대해 TRUE 또는 FALSE 값이 필요 `Wait` 합니다. 다음 코드 예제에서는 이 경고를 생성합니다.  
  
```  
ExAcquireResourceExclusiveLite(Resource, Wait);  
```  
  
 다음 코드 예제에서는 이 경고를 방지합니다.  
  
```  
ExAcquireResourceExclusiveLite(Resource, TRUE);  
```
