# <a name="setter-functions-must-have-one-argument"></a>setter 함수는 하나의 인수를 사용해야 합니다.
1051: Setter 함수는 하나의 인수를 포함 해야 합니다.  
  
 Setter 함수에는 인수가 하나만 있어야 합니다.
