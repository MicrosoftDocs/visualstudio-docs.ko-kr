# <a name="spy-menu-commands"></a>Spy 메뉴 명령
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

**Spy** 메뉴는 Spy++ 창과 뷰에서 사용할 명령을 제공합니다.  
  
 **Windows**  
 새 창 뷰를 엽니다.  
  
 **프로세스**  
 새 프로세스 뷰를 엽니다.  
  
 **스레드**  
 새 스레드 뷰를 엽니다.  
  
 **로그 메시지**  
 새 메시지 보기를 열고 보기의 [메시지 옵션 대화 상자](../debugger/message-options-dialog-box.md) 를 엽니다.  
  
 **창 찾기**  
 마우스를 사용 하 여 창을 선택 하는 **FindWindow** 대화 상자를 표시 합니다.  
  
 **끝내기**  
 Spy + + 유틸리티를 닫습니다.
