# <a name="idiasymbolget_haslongjump"></a>IDiaSymbol::get_hasLongJump
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

함수에 [setjmp](https://msdn.microsoft.com/library/684a8b27-e8eb-455b-b4a8-733ca1cbd7d2) 명령을 사용 [하 여이 명령에](https://msdn.microsoft.com/library/0e13670a-5130-45c1-ad69-6862505b7a2f) 대 한 사용이 포함 되는지 여부를 지정 하는 플래그를 검색 합니다 .이는 예외 처리의 C 스타일 메서드를 형성 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp#  
HRESULT get_hasLongJump  
   BOOL *pFlag  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pFlag`  
 제한이 `TRUE` 함수에 명령이 있으면를 반환 `longjmp` 하 고, 그렇지 않으면를 반환 `FALSE` 합니다.  
  
## <a name="return-value"></a>반환 값  
 성공 하면이 반환 되 `S_OK` 고, 그렇지 않으면 `S_FALSE` 또는 오류 코드가 반환 됩니다.  
  
> [!NOTE]
> 반환 값은 `S_FALSE` 기호에 대해 속성을 사용할 수 없음을 의미 합니다.
