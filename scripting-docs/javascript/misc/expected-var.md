' Var ' (ERRnoVar 1034)이 필요 합니다.  
  
 필요한 경우 var 키워드를 추가 하지 못했습니다.  
  
 Var 키워드에 대 한 자세한 내용은 [Var 문](../../javascript/reference/var-statement-javascript.md)을 참조 하십시오.
