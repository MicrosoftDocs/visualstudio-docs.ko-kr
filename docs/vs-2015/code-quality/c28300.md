# <a name="c28300"></a>C28300
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C28300: < parameter_name >: 연산자에 호환 되지 않는 형식의 식 피연산자 < operator_name >  
  
 이 경고는 매개 변수에서 피연산자 형식이 연산자와 호환되지 않을 때 보고됩니다.
