# <a name="evaluate-statement-command"></a>문 실행 명령

지정된 문을 평가 및 표시합니다.

## <a name="syntax"></a>구문

```cmd
>Debug.EvaluateStatement text
```

## <a name="arguments"></a>인수

`text`

필수 요소. 평가할 문입니다.

## <a name="example"></a>예제

```cmd
>Debug.EvaluateStatement args.Length
```
