# <a name="cores-view-logical-cores"></a>코어 뷰 논리 코어
각 채널 이름은 프로파일링된 시스템에 존재하는 논리 코어 중 하나를 나타냅니다. 명명은 0에서 논리 코어의 총 수까지 셉니다. 예를 들어 4코어 시스템은 논리 코어 0에서 논리 코어 3까지 셉니다.
