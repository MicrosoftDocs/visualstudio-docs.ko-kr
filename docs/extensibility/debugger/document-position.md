# <a name="document-position"></a>문서 위치
[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]디버깅할 때 *문서 위치는* 입니다.

- IDE에 알려진 소스 파일의 위치를 추상화합니다. 오늘날 대부분의 언어에서 문서 위치는 원본 파일의 위치로 간주될 수 있습니다.

- 소스 문서에서 디버그 엔진의 위치를 설명합니다.

- [IDebugDocumentPosition2](../../extensibility/debugger/reference/idebugdocumentposition2.md) 인터페이스에 의해 구현됩니다.
