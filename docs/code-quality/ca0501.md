# <a name="ca0501"></a>CA0501
출력 보고서를 읽을 수 없습니다.

다음 이유 중 하나로 인해 출력 보고서를 쓸 수 없는 경우 발생 합니다.

- 디스크가 꽉 참

- 디렉터리에 쓸 수 있는 권한이 없습니다.

- 디렉터리가 없습니다.
