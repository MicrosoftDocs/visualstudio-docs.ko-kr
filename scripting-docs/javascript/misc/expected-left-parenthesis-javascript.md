# <a name="expected--javascript"></a>'('가 필요합니다.(JavaScript)
괄호 안에 식을 묶어야 했지만 여는 괄호를 포함 하지 않았습니다. 일부 식은 여는 괄호와 닫는 괄호 집합 안에 포함 되어야 합니다. 다음 예제에서는 괄호를 사용 합니다.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 왼쪽 괄호를 계산 식에 추가 합니다.
