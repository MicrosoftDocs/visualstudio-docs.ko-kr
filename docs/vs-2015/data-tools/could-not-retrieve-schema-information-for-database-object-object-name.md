# <a name="could-not-retrieve-schema-information-for-database-object-ltobject-namegt"></a>데이터베이스 개체 개체 이름에 대 한 스키마 정보를 검색할 수 없습니다. &lt;&gt;
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

이 메시지는 일반적으로 **서버 탐색기/데이터베이스 탐색기** 의 개체를 클립보드로 복사 하 고 데이터베이스에서 삭제 한 다음 디자이너에 붙여 넣을 때 나타납니다. 즉, 데이터베이스 개체가 더 이상 존재하지 않으므로 이 메시지가 나타납니다.

### <a name="to-correct-this-error"></a>이 오류를 해결하려면

- **확인**을 클릭하여 대화 상자를 닫습니다.
