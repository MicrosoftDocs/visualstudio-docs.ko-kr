# <a name="ca0056"></a>CA0056

프로젝트 파일에 잘못 된 버전의 분석 도구가 있습니다.

이 경고는 잘못 된 코드 분석 도구 버전이 있는 프로젝트 파일을 사용 하 여 FxCopCmd.exe를 수동으로 호출 하는 경우에 발생 합니다.
