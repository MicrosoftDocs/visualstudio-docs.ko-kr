# <a name="expected-constant"></a>상수가 필요합니다.
조건부 컴파일 테스트 문에서 (비 조건부 컴파일) 변수를 사용 하려고 했습니다. 조건부 컴파일 테스트 문은 상수로 계산 되어야 합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 변수를 리터럴로 바꿉니다.  
  
- 변수를 조건부 컴파일 변수로 바꿉니다.
