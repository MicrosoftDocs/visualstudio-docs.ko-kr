# <a name="ca0069"></a>CA0069

UsingAlternatePlatform. *FrameworkVersion1* 를 찾을 수 없습니다. 대신 *FrameworkVersion2* 를 사용 합니다. 최상의 분석 결과를 위해 올바른 .NET Framework 설치 되어 있는지 확인 하십시오.
