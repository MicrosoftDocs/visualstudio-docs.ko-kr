# <a name="notifications-environment-options-dialog-box"></a>알림, 환경, 옵션 대화 상자

이전에 **알림** 대화 상자를 통해 무시하도록 선택한 알림에 대해 무시를 중지하려면 이 옵션을 사용합니다. 자세한 내용은 [Visual Studio 업데이트](../../install/update-visual-studio.md)를 참조하세요.
