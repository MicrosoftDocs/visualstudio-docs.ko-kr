# <a name="globalizing-applications"></a>애플리케이션 전역화
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

전역화는 여러 문화권에서 작동하도록 소프트웨어를 디자인하고 개발하는 프로세스입니다. 이 섹션은 Windows Forms 및 Web Forms 페이지에 둘 다 적용됩니다.

## <a name="in-this-section"></a>섹션 내용
 [전역 Windows Forms 및 Web Forms에 대 한 문화권별 클래스](../ide/culture-specific-classes-for-global-windows-forms-and-web-forms.md) 문화권 설정에 따라 날짜, 시간, 숫자, 통화 및 기타 정보의 형식을 지정 하는 클래스에 대해 설명 합니다.
