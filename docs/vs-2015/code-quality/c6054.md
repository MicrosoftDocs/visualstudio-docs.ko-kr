# <a name="c6054"></a>C6054
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C6054: 문자열 \<variable> 을 0으로 종료할 수 없습니다.  
  
 이 경고는 0으로 종료 된 문자열이 필요한 함수에 0이 아닌 종료 된 문자열이 전달 되었음을 나타냅니다. 0으로 끝나는 문자열이 필요한 함수는 0을 찾기 위해 버퍼의 끝을 넘어 갑니다. 이 오류로 인해 악용 가능한 버퍼 오버런 오류 또는 충돌이 발생할 수 있습니다. 프로그램은 문자열이 0으로 끝나는지 확인 해야 합니다.  
  
## <a name="example"></a>예  
 다음 코드에서는이 경고를 생성 합니다.  
  
```cpp  
  
#include <sal.h>  
  
void func( _In_z_ wchar_t* wszStr );  
  
void g ( )  
{  
    wchar_t wcArray[200];  
    func(wcArray); // Warning C6054  
}  
```  
  
 이 경고를 해결 하려면 `wcArray` `func()` 다음 샘플 코드에 표시 된 것 처럼 함수를 호출 하기 전에 null을 종료 합니다.  
  
```cpp  
  
#include <sal.h>  
  
void func( _In_z_ wchar_t* wszStr );  
  
void g( )  
{  
    wchar_t wcArray[200];   
    wcArray[0]= '\0';  
    func(wcArray);  
}  
```
