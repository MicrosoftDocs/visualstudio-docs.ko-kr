# <a name="exception-assistant-dialog-box"></a>예외 도우미 대화 상자
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

런타임 예외가 throw 되 면 **예외 도우미** 대화 상자가 나타납니다. 예외 **도우미** 는 예외의 유형을 표시 하 고, 추가 정보를 제공 하 고, 문제 해결 팁에 대 한 링크를 제공 하 고, 온라인으로 추가 도움말을 검색 하는 방법을 제공 하며, 사용자가 예외의 세부 정보를 보는 등의 특정 작업을 수행할 수 있도록 합니다.  
  
## <a name="task-list"></a>작업 목록  
 발생 한 예외 유형에 대 한 문제 해결 관련 항목을 보려면 **문제 해결 팁** 창에 표시 된 팁 메시지 중 하나를 클릭 합니다.  
  
 예외와 관련된 작업을 수행하려면 작업 창에 표시된 작업 중 하나를 클릭합니다.  
  
 예외 도우미를 사용 하거나 사용 하지 않도록 설정 하는 방법에 대 한 자세한 내용은 [옵션 대화 상자, 디버깅, 일반](../debugger/general-debugging-options-dialog-box.md)을 참조 하세요.  
  
## <a name="user-interface-element-list"></a>사용자 인터페이스 요소 목록  
 **예외 형식**  
 throw된 예외의 형식을 표시합니다.  
  
 **추가 정보**  
 예외에 대한 추가 정보를 표시합니다.  
  
 **문제 해결 팁**  
 예외의 원인을 발견하는 데 도움이 될 수 있는 문제 해결 팁에 대한 링크를 표시합니다.  
  
 **actions**  
 예외 개체에 대한 추가 정보 표시를 비롯하여 수행할 수 있는 작업을 나열합니다.  
  
 **온라인 도움말 보기**  
 온라인으로 추가 도움말을 검색할 수 있습니다.
