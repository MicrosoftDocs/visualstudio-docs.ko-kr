# <a name="invalid-number"></a>숫자가 잘못되었습니다.
잘못 된 숫자 (ErrbadNumber 1013)  
  
 JavaScript에서 잘못 된 숫자를 사용 하려고 했습니다.  
  
 유효한 숫자에 대 한 자세한 내용은 [Number 개체](../../javascript/reference/number-object-javascript.md)를 참조 하세요.
