# <a name="the-use-of-a-keyword-for-an-identifier-is-invalid"></a>식별자에 대해 키워드를 잘못 사용했습니다.
1048: 식별자에 대 한 키워드를 사용 하는 것이 잘못 되었습니다.  
  
 키워드는 변수, 함수 또는 기타 사용자 정의 식별자의 이름으로 사용할 수 없습니다.  
  
 키워드에 대 한 자세한 내용은 [Javascript 예약어](../../javascript/reference/javascript-reserved-words.md) 및 [javascript Future 예약어](../../javascript/reference/javascript-future-reserved-words.md)를 참조 하세요.
