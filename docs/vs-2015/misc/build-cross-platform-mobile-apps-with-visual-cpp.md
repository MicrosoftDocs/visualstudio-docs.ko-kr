# <a name="build-cross-platform-mobile-apps-with-visual-c"></a>Visual C++를 사용하여 플랫폼 간 앱 빌드
플랫폼 간 모바일 개발용 Visual C++를 사용하여 Android, iOS 및 Windows 디바이스용 플랫폼 간 코드를 빌드할 수 있습니다. 이는 Visual Studio 2015에서 사용할 수 있는 선택적 기능으로, Visual C++를 사용하여 iOS, Android 및 Windows용 플랫폼 간 코드를 개발할 수 있게 해줍니다.  
  
 설치 지침 및 코드 예제는 [플랫폼 간 모바일 개발용 Visual C++](../cross-platform/visual-cpp-for-cross-platform-mobile-development.md)를 참조 하세요.
