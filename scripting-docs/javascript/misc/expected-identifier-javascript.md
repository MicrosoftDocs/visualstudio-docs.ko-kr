# <a name="expected-identifier-javascript"></a>식별자가 필요합니다.(JavaScript)
필요한 컨텍스트에서 식별자 이외의 항목을 사용 했습니다. 식별자는 다음과 같을 수 있습니다.  
  
- 변수,  
  
- 속성,  
  
- 배열,  
  
- 또는 함수 이름입니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 식별자가 등호 왼쪽에 표시 되도록 식을 변경 합니다.
