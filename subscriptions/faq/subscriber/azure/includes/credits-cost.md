## <a name="how-much-does-it-cost"></a>가격은 얼마인가요?

크레딧 사용 비용은 없습니다. 크레딧은 Visual Studio 구독에 포함된 사항입니다. 해당 월에 할당된 크레딧을 모두 사용하면, 다음 달에 초기화될 때까지 크레딧을 계속 사용할 수 없습니다.
