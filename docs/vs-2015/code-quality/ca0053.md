# <a name="ca0053"></a>CA0053

규칙 어셈블리를 로드할 수 없습니다.

이는 리플렉션을 통해 규칙 어셈블리를 로드할 수 없음을 나타냅니다. 이는 사용자가 올바르게 빌드되지 않은 사용자 지정 규칙을 사용 하거나 분석 도구에서 파일 시스템 액세스가 거부 되는 등의 이유로 어셈블리를 로드할 수 없는 경우에 발생할 수 있습니다.
