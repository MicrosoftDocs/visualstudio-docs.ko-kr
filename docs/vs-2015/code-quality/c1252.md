# <a name="c1252"></a>C1252
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C1252: 플러그 인 간에 순환 또는 누락 종속성이 있습니다. GUID가 필요 합니다.  
  
 코드 분석 도구는 분석 중인 코드가 아니라 플러그 인 종속성에 내부 오류가 있는 경우에 이 경고를 보고합니다.
