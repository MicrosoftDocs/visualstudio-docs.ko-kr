# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C28225: 주석에 필요한 매개 변수 개수가 잘못 되었습니다.  
  
 이 경고는 명명 된 주석이 잘못 된 수의 매개 변수와 함께 사용 됨을 나타냅니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
