# <a name="common-data-views"></a>공통 데이터 뷰
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

이 섹션에는 모든 프로파일링 방법에 공통적인 프로파일러 데이터 파일의 뷰와 보고서에 대한 참조 정보가 포함되어 있습니다.
