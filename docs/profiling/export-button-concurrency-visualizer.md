# <a name="export-button-concurrency-visualizer"></a>내보내기 단추(동시성 시각화 도우미)
**내보내기** 단추를 통해 사용자 고유의 레코드 또는 Microsoft Excel과 같은 다른 도구와 함께 사용하기 위해 .*csv* 파일로 호출 스택을 내보낼 수 있습니다.
