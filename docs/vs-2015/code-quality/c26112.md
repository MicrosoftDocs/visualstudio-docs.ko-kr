# <a name="c26112"></a>C26112
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C26112: 호출자가 호출 하기 전에 잠금을 유지할 수 없습니다 \<func> .  
  
 주석은 `_Requires_no_locks_held_` 호출자가 함수를 호출 하는 동안 잠금을 보유 하지 않아야 하는 사전 조건을 적용 합니다. 경고 C26112는 다른 함수를 호출 하기 전에 함수가 모든 잠금을 해제 하는 데 실패 한 경우에 발생 합니다.  
  
## <a name="example"></a>예  
 다음 예제에서는 `_Requires_no_locks_held_` 잠긴 섹션 내에 대 한 호출에서 사전 조건을 위반 하기 때문에 경고 C26112을 생성 합니다 `NoLocksAllowed` .  
  
```  
  
typedef struct _DATA   
{  
    CRITICAL_SECTION cs;  
} DATA;  
  
_Requires_no_locks_held_   
  
void NoLocksAllowed(DATA* p)  
{  
     // Lock sensitive operations here  
}  
  
void LocksHeldFunction(DATA* p)   
{   
    EnterCriticalSection(&p->cs);   
    NoLocksAllowed(p); // Warning C26112  
    LeaveCriticalSection(&p->cs);  
}  
  
```
