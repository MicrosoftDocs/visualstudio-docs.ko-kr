# <a name="visual-language-dictionary"></a>시각적 언어 사전

Visual Studio 공통 시각적 언어를 사용하여 아이콘 및 기타 이미지의 구성 요소인 개념을 설명합니다. 새 아이콘을 디자인할 때 이 사전을 사용하여 사용자 고유의 이미지에 전달하려는 개념을 식별합니다.

## <a name="general-guidelines"></a>일반 지침

- 이러한 시각적 개념은 사전 정의와 함께 일관되게 사용해야 합니다. 그렇지 않으면 공통 요소가 일관된 의미를 가질 것으로 예상하는 사용자를 혼동할 위험이 있습니다.

- 이미지가 선명하고 구분하기 쉽도록 아이콘에서 사용할 가장 관련성이 있거나 중요한 개념만 선택합니다.

- 각 이미지에서 하나 또는 두 개의 개념만 사용합니다.

- 아이콘 디자인에 대한 자세한 전략은 [아이콘 디자인을](../../extensibility/ux-guidelines/images-and-icons-for-visual-studio.md#BKMK_IconDesign)참조하세요.
