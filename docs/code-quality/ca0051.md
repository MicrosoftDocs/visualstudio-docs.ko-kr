# <a name="ca0051"></a>CA0051

규칙을 선택 하지 않았습니다.

이 오류는 관리 코드 분석 도구를 사용할 수 있지만 규칙이 선택 되지 않았음을 나타냅니다. 이 오류를 해결 하려면 적절 한 규칙을 선택 합니다.
