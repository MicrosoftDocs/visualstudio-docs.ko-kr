# <a name="ca0502"></a>CA0502

지원 되지 않는 언어

이 오류는 사용자가 c # 또는 Visual Basic 이외의 언어로 작성 된 하나 이상의 페이지나 클래스를 포함 하는 ASP.NET 프로젝트를 분석 하려고 할 때 발생 합니다. 이 오류를 해결 하려면 프로젝트에서 파일을 제거 하거나 코드 분석을 사용 하지 않도록 설정 합니다.
