이 메시지는 일반적으로 **서버 탐색기 Explorer** 또는 **데이터베이스 탐색기**에서 개체를 클립보드에 복사하고 데이터베이스에서 삭제한 다음, 디자이너에 붙여넣은 경우에 나타납니다. 즉, 데이터베이스 개체가 더 이상 존재하지 않으므로 이 메시지가 나타납니다.
