 블록에 포함 되지 않아야 하는 형식이 함수에 있는 경우이 경고가 보고 됩니다 `try/except` `try/except` . 함수가 호출한 하나 이상의 경로에서 발견된 코드 분석 도구가 `try/except` 블록 안에 있습니다.
