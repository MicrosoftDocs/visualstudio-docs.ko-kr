# <a name="ca0057"></a>CA0057

위반을 현재 대상 및 규칙 집합에 매핑할 수 없습니다.

이 오류는 현재 대상 및 규칙 집합에 매핑할 수 없는 위반이 있는 프로젝트 파일 또는/가져온 XML 보고서를 사용 하 여 FxCopCmd.exe를 수동으로 호출 하는 경우에 발생 합니다.

도구의 명령줄 버전을 사용 하 여 분석을 수행 하는 경우에만 발생 합니다.
