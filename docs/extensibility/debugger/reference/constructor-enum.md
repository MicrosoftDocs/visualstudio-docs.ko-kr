# <a name="constructor_enum"></a>CONSTRUCTOR_ENUM
다른 형식의 생성자를 선택 합니다.

## <a name="syntax"></a>Syntax

```cpp
typedef enum ConstructorMatchOptions {
    crAll       = 0,
    crNonStatic = 1,
    crStatic    = 2
} CONSTRUCTOR_ENUM;
```

```csharp
public enum ConstructorMatchOptions {
    crAll       = 0,
    crNonStatic = 1,
    crStatic    = 2
};
```

## <a name="fields"></a>필드
`crAll`\
모든 생성자를 선택 합니다.

`crNonStatic`\
비정적 생성자를 선택 합니다.

`crStatic`\
정적 생성자를 선택 합니다.
