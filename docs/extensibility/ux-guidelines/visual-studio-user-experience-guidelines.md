# <a name="visual-studio-user-experience-guidelines"></a>Visual Studio 사용자 환경 지침
Visual Studio의 새로운 기능을 디자인하는 경우 Visual Studio 사용자 환경 지침을 따릅니다. 해당 지침에는 새로운 UI(사용자 인터페이스)의 디자이너가 개발 환경 내에서 원활하고 일관된 사용자 환경을 만들 수 있도록 일반적인 사용자 모델 및 상호 작용 패턴에 관한 정보가 포함됩니다.

Microsoft 제품용 소프트웨어를 개발하는 경우 다음 리소스를 사용하는 방법을 알아봅니다.

- [Windows 사용자 환경 상호 작용 지침](/windows/win32/uxguide/guidelines)은 Windows 데스크톱 동작 및 모양에 관한 기본 사항입니다.

- 이 Visual Studio 지침은 환경에 적절한 모든 경우에 일반적인 Windows 지침을 대체하는 플랫폼별 지침입니다.

- [UWP 앱의 컨트롤 및 패턴](/windows/uwp/design/controls-and-patterns)은 Visual Studio 내의 일부 인스턴스에서 사용할 수 있는 새로운 패턴의 참조입니다.

해당 지침은 Visual Studio를 위해 디자인할 때 수행할 작업에 관한 정보를 제공하며, 경우에 따라 이 작업을 수행하는 방법에 관한 정보도 포함됩니다. 더 자세한 구현 정보를 확인하고 Visual Studio IDE(통합 개발 환경)에 통합되는 솔루션에 대한 코드를 작성할 수 있는 특정 API(애플리케이션 프로그래밍 인터페이스)에 관해 알아보려면 [Visual Studio SDK(소프트웨어 개발 키트)](../visual-studio-sdk.md)를 사용합니다.
