# <a name="monitor-with-tensorboard"></a>TensorBoard를 사용하여 모니터링

TensorBoard를 사용하여 모델 학습 진행률을 시각화합니다.

1. 프로젝트를 마우스 오른쪽 단추로 클릭하고 **TensorBoard 실행** 을 클릭한 다음, 출력 TensorBoard 로그의 디렉터리를 선택합니다.

    ![MNIST 프로젝트가 선택된 Visual Studio 솔루션 탐색기의 스크린샷 상황에 맞는 메뉴가 열려 있고 TensorBoard 실행 명령이 선택되어 있습니다.](media/monitor-tensorboard/run-tensorboard.png)

2. 시간이 지나면 오류가 감소합니다. 즉, 성능이 개선됩니다.

    ![TensorBoard 로그 데이터의 그래픽 시각화를 보여 주는 주 TensorBoard 창의 스크린샷](media/monitor-tensorboard/tensorboard.png)
