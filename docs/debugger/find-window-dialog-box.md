# <a name="find-window-dialog-box"></a>창 찾기 대화 상자
이 대화 상자는 창을 선택하고 해당 속성 또는 메시지를 표시하는 데 사용됩니다. 이 대화 상자를 표시하려면 **Spy** 메뉴에서 **창 찾기**를 선택하세요.

 다음 설정을 사용할 수 있습니다.

 **찾기 도구** 창을 선택하는 데 사용되는 끌어서 놓기 도구입니다.

 **Spy++ 숨기기** 주 Spy++ 창을 제거하고 **창 찾기** 대화 상자가 다른 애플리케이션 위에 표시되도록 유지합니다.

 **핸들** 찾을 창의 고유 ID입니다.

 **캡션** 창 캡션의 텍스트입니다.

 **클래스** 창의 클래스입니다.

 **스타일** 창의 스타일입니다.

 **사각형** 창의 경계 사각형입니다.

 **속성 또는 메시지 표시** 확인을 눌러 [창 속성 대화 상자](../debugger/window-properties-dialog-box.md)를 표시할 것인지 [Messages View](../debugger/messages-view.md)(메시지 뷰)를 열 것인지 결정합니다.
