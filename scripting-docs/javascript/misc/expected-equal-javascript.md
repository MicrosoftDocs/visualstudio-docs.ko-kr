# <a name="expected--javascript"></a>'='가 필요합니다.(JavaScript)
조건부 컴파일 문에서 사용할 변수를 만들려고 했지만 변수와이 변수에 할당 하려는 값 사이에 등호를 두지 않았습니다 ().  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 등호를 추가 합니다. 예를 들어:  
  
    ```JavaScript  
    @set @myvar1 = 1  
    ```
