# <a name="stepunit"></a>STEPUNIT
단계별 실행 단위를 지정 합니다.

## <a name="syntax"></a>구문

```cpp
enum enum_STEPUNIT { 
   STEP_STATEMENT   = 0,
   STEP_LINE        = 1,
   STEP_INSTRUCTION = 2
};
typedef DWORD STEPUNIT;
```

```csharp
enum enum_STEPUNIT { 
   STEP_STATEMENT   = 0,
   STEP_LINE        = 1,
   STEP_INSTRUCTION = 2
};
```

## <a name="fields"></a>필드
 `STEP_STATEMENT`\
 문 별 단계.

 `STEP_LINE`\
 단계별 설명입니다.

 `STEP_INSTRUCTION`\
 단계별 지침을 설명 합니다.

## <a name="remarks"></a>설명
 [단계](../../../extensibility/debugger/reference/idebugprocess3-step.md) 메서드에 인수로 전달 됩니다.

 헤더: msdbg .h

 네임 스페이스: VisualStudio

 어셈블리: Microsoft.VisualStudio.Debugger.Interop.dll
