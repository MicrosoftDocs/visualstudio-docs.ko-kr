# <a name="expected-var"></a>'var'이 필요합니다.
' Var ' (ERRnoVar 1034)이 필요 합니다.  
  
 필요한 경우 var 키워드를 추가 하지 못했습니다.  
  
 Var 키워드에 대 한 자세한 내용은 [Var 문](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Statements/var)을 참조 하십시오.
